Al usar TestimonioYa, aceptas estos términos. Si no estás de acuerdo, no uses el servicio.

2. Descripción del Servicio

TestimonioYa es una plataforma para recopilar, gestionar y mostrar testimonios de clientes.

3. Cuentas

  • Debes proporcionar información veraz
  • Eres responsable de tu contraseña
  • Una cuenta por negocio (plan gratuito)

4. Uso Aceptable

No puedes usar el servicio para contenido ilegal, spam, testimonios falsos o actividades fraudulentas.

5. Pagos y Cancelación

  • Los planes de pago se facturan mensualmente

Los testimonios son propiedad de los usuarios que los crean. TestimonioYa mantiene los derechos sobre la plataforma.

7. Limitación de Responsabilidad

El servicio se proporciona "tal cual". No garantizamos disponibilidad ininterrumpida.
